# APNA SCHOOL - School Management System

A modern, responsive school management portal built with HTML, CSS, and JavaScript.

## üåü Features

- **Modern UI**: Glassmorphism design, vibrant colors, and responsive layout.
- **Hero Carousel**: Interactive 3D carousel on the home page.
- **Student Portal**: View marks, attendance, and profile.
- **Admin Panel**: 
  - Login authentication (Default: `admin` / `admin123`)
  - Add/Manage Students
  - Add/Manage Marks
  - Generate Mark Sheets (Printable format)
  - View Statistics (Pass percentage, Top performers)
- **Local Storage**: Data persistence using browser LocalStorage (no backend required).

## üöÄ Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Edge).
- Node.js (optional, for running local server).

### Run Locally

1. **Clone or Download** the repository.
2. **Open the project folder** in your terminal.
3. **Start a local server**:
   You can use `http-server` via npx:
   ```bash
   npx -y http-server -p 8000
   ```
4. **Access the site**:
   Open `http://localhost:8000` in your browser.
